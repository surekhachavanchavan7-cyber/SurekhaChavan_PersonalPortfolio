# Personal Portfolio – Surekha Chavan

## Overview

A responsive personal portfolio built with HTML, CSS, and JavaScript to showcase my skills, projects, and resume.

## Sections

- **Home** – Intro + professional portrait
- **About** – Education, experience, skills
- **Projects** – 3 sample projects with images and descriptions
- **Contact** – Form with JavaScript validation
- **Resume** – Opens `resume.pdf` in a new tab

## Technologies

HTML5 • CSS3 (Responsive) • JavaScript (DOM, validation, IntersectionObserver)

## How to Run

1. Place images inside `/assets` and your resume as `resume.pdf` in the project root.
2. Open `index.html` in a browser (or use VS Code “Live Server”).

## Notes

- Contact form demonstrates **client-side validation only** (no backend).
- Navigation has **smooth scrolling** and **active link** highlighting.
